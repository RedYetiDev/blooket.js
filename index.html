Basic Information

The Blooket.JS code is a class. so when you run 
              
                const Blooket = require("blooket.js")
                // mainly the following part
                game = new Blooket()
variables are created. The important ones are game.pin, game.name, game.questions, and if you are in Gold Quest, game.prizes. So if I were to log "game", it would return a list of all the diffent variables attached.

Advanced setup

The blooket package uses EventEmitter to emit different events like GameStart or QuestionStart. So, for more advanced setup, you can catch these events and run code when they are received.

          
            //start with the basic code
            Blooket = require("blooket.js")
            game = new Blooket()
            game.join(pin, name)
            // now we add the advanced code
            // The events you can use are
            // GameStart,
            // QuestionStart,
            // QuestionAnswered.
            // Most of them return game.
            // To use events, use this command
            game.on("Event", function(data) {

            })
            // That is saying when the game is sending "Event", run a function with the data
